# LrNCHeMnOW
LrNCHeMnOW is a educational app and database for chemistry, featuring many different functionalities!

Please read LICENSE.txt before using the application

The .html version of the app comes as a standalone and works on any device with a web browser. The app is not intended for mobile users, even if most features will still work.
The OS-specific version of the app come in as Electron-packaged versions. You will need to read the specific instructions on how to use the app.
In most cases, the .html version is your best choice. Easiest, works everywhere.
